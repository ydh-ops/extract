중간점검 실무 매뉴얼

빠르고 정확한 서류 검토를 위한 스텝 바이 스텝 가이드

1. 사전 준비

검토 전 준비 서류

중간점검을 시작하기 전에 아래의 3가지 서류를 준비(확인)해 주세요.

* 주의: 파일 링크 수정은 메모장 등으로 index.html을 열어 최상단 설정(CONFIG)을 변경해 주세요.

2. 기업정보 검토사항

주요 대조 확인 내용

  • 운영사명 / 기업명 / 대표자명 :
    수정사업계획서의 내용과 완벽히 동일한지 확인
  • 지원구분 / 법인 계속 유지 여부 / 분야 :
    체크 여부 확인 및 수정사업계획서 내용과 일치 여부
    (예: 스케일딥 / 스케일업 등)
  • 주업종 : 수정사업계획서의 주업종과 동일한지 확인
    (※사업자등록증과도 동일한지 대조)
  • 아이템명 : 수정사업계획서의 "사업아이템명"과 동일한지 확인
    ( 사업아이템소개X )

3. 사업비 집행 검토사항

집행 금액 대조표 검토

사업시작일로부터 기준일의 총 사업비 , 정부지원금 , 자기부담금(현금+현물) 내역을 확인합니다.

확인 지표 예산, 집행금액, 집행잔액, 집행률(%)
핵심 검토사항 사업비 집행 내용이 질링스 사업비 집행 내용과 동일한지
반드시 확인합니다.
참고 파일 [질링스 데이터 및 사업비 집행률.xlsx] 파일 대조

4. 주요 추진실적 검토사항

실적 지표 (3개월 기준)

  • 항목별 주요 추진실적 내용이 질링스 데이터와 동일한지 대조
  • 매출액 : 국내 매출액을 기준으로 확인
  • 수출액 : 해당 기간 내 수출 실적 확인
  • 고용 : 협약 시점 인원, 3개월 후 인원 동일한지 확인
    신규 고용 인원 확인
  • 투자 : (투자처), (투자금액), (투자 계약일) 해당 양식에 맞게 작성했는지
  • 지재권 : 출원 및 등록 건수(00건) 확인

5. 사업운영 실적 검토사항

추진 내용 및 목표 달성 검토

  • 수정사업계획서의 추진 내용과 동일한지 :
    ③ 달성목표 및 사업추진 계획 -> 협약기간 내 목표 및 달성 계획 -> 추진 내용 항목 대조
  • 세부 추진 실적 및 향후 추진 계획 이 알맞게 작성되었는지 논리적 검토

6. 사업비 구성 및 집행 현황 (!매우 중요)

가장 빈번하게 오류가 발생하는 항목입니다.

  • 예산 구성 일치 : 사업비 예산 구성이 질링스 데이터와 동일한지,
    비율 을 알맞게 표기했는지 점검
  • 수정사업계획서 일치 : 수정사업계획서의 예산안과 동일한지 대조
  • 집행률 검토 : 합계 금액 및 합계 집행률 이 제대로 계산되었는지 확인
  • 세목별 분리 : 집행내용/예산 항목을 세목별로 각각 기재하였는가?
  • 1페이지 매칭 : 1페이지의 "사업비 집행" 섹션에 적힌
    집행 금액 및 집행률 과 일치하는가?
  • 부진 사유 : 집행률이 낮은 경우 집행 부진 사유가 작성되었는지 확인

7. 협약 변경 사항 검토사항

협약 변경 이력 검토

  • 협약 변경 내용이 양식에 맞게 알맞게 작성되었는지 확인
  • 변경 사항이 없을 경우 , 공란으로 두지 않고 “해당 내용 없음” 이라고
    작성하였는지 필수 확인
